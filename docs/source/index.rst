polyview overview
======================

*polyview* is a sklearn-compatible multi-view learning toolkit. It provides a range of multi-view learning algorithms, including clustering, dimensionality reduction, fusion methods, data augmentation methods, and more.

This documentation includes:

- A brief introduction to the package
- A complete API reference
